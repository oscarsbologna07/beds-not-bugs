# Beds Not Bugs

Calculate hospital bed utilization by comparing state-level hospital bed
data to COVID-19 hospitalization data from FDOH.

## Installation

You'll need the Ruby version listed in `.ruby-version` to be installed for
this script to run.

## Usage

- `ruby bedsnotbugs.rb`
